# [分析] レベルデザイン（ステージ構成）

このページでは[](スキルカード)を組み立ててカリキュラムのレベルデザインを行います。

```{figure} https://i.gyazo.com/e4b5dd3cd81241196879631d0e95b07a.png
- 各レベルには冒頭部に概要・前提条件・目的を持つように設計します。
- また、次のステージ卒業の試練（ボス戦）があります。
- ステージをクリアーしたら、キャラクターシートを鉛筆で塗りつぶします。
```

なお、学習者スキルを表現するために、[](キャラクターシート)では、「現在のステージ(このステージで設計します)」と「スキル毎のレベル([](スキルカード)のページで設計します)」で表現します。

ステージは一本道の構成です。
