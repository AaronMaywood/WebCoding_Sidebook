##### 名前を付ける機能について

`HTML`の要素には製作者が名前を付けることのできる機能が２種類あります。

1. まず、{bdg-dark-line}`テキスト：P.82` で説明する「クラス名」です。（「[](div-span)」でも紹介しました）
  クラスは「学級（生徒をグループ分けしたもの）」と同じ意味で、要素をグループ分けするのに用います。
  次のように書いたら、
  ```html
  <h1 class="a">見出し１</h1>
  <p class="a">段落１</p>
  <p class="b">段落２</p>
  <p>段落３</p>
  <h2 class="a">見出し２</h2>
  <p class="b">段落４</p>
  ```
  「aクラス」に所属するのは「見出し１」「段落１」「見出し２」で、「bクラス」に所属するのは「段落２」「段落４」です。（「段落３」はどのクラスにも所属していません。）

2. 次に、{bdg-dark-line}`テキスト：P.83`にある `ID`名です。 % https://www.weblio.jp/content/%E3%82%A2%E3%82%A4%E3%83%87%E3%82%A3%E3%83%BC
  `ID`は`identification`の略で、俗に「IDカード」などという個人を特定する身分証明書を意味する単語です。
  名前を付けるという意味ではクラスと同じですが、クラスが複数の要素に同じ名を付けてグループ分けすることができたのとは対象的に、個の識別名である`ID`は同じ名前を同じページ内で２度以上使うことができません。
  ```html
  <p id="a">段落１</p>
  <p id="a">段落２</p> <!-- 同じ名前を２度使うことはできません！ -->
  <p id="b">段落４</p>
  ```
  `ID`名は常に特定の一つの要素を指す名前として使用します。

##### スタイルの優先順位について

{bdg-dark-line}`テキスト：P.82` には、「HTML内で同じクラス名を複数のあちこちで使用可能で、通常要素（タイプセレクタ）の**約１０倍強く作用します**。」という説明があります。これは「スタイルには優先順位がある」ということを説明しています。

`CSS`の基本的な動作は、**同じセレクターに対するスタイルがあったとき、後の方に書いたスタイル**が優先されます。

次の`HTML`をもとに説明します。
```HTML
<p>段落１</p>
<p class="test">段落２</p>
```

次の`CSS`では、`p`要素の色は`green`となります。（最初の`red`の指定が上書きされて解釈されます）
```CSS
p {
  color: red;
}

p {
  color: green;
}
```

しかし、クラスセレクタを付与すると「約10倍強い」優先度となり、`class="test"`が指定された「段落２」の文字色は`red`となります。（「段落１」にはクラス名がないので`green`のままです。）
```CSS
/* "p.test" とは、p要素かつクラス名が"test"であるものを指します。なお、".test" というようにクラス名だけを指定することも可能です。 */
p.test {
  color: red;
}

p {
  color: green;
}
```

これが`CSS`の優先順位です。

優先順位のおおざっぱな理解は次のとおりです。
- 通常の要素（例えば`p`）よりも、わざわざ製作者がクラス名を付けた要素の方が優先度が高くなるようになっています。（ありふれた`p`は`green`だが、`testクラス`に所属している`p`は特別に`red`というように）
  - 更に、`ID`名は固有名称であるため、最も優先度が高くなります。（ありふれた`p`は`green`、`testクラス`に所属している`p`だけは`red`、しかし`taro-yamada`だけは`blue`のように）

## 色

CSSで色を指定するにはいくつかの方法があります。

1. `red`, `green`等の色名を使う
2. 光の３原色(RGB=赤、緑、青)で指定する
  a. 16進法を使って指定する
  b. 10進法を使って指定する→`rgb()`を使います→併せて透明度も指定できる`rgba()`もあります
3. （テキストの範囲外に、`rgb()`よりも直感的に色指定ができる`hsb()`などの方法があります。）

Cascading Style Sheets（CSS、カスケーディング・スタイル・シート、カスケード・スタイル・シート）は、HTML や XML の要素をどのように修飾（表示）するかを指示する仕様の一つで、World Wide Web Consortium (W3C) がとりまとめ勧告する。文書の構造と体裁を分離させるという理念を実現するために提唱されたスタイルシートの具体的な仕様の一つ。

記述方法
p#id { color: #ff3300 }

上例は HTML 文書に適用する場合、「 id という ID を持った p 要素の文字色を赤 FF(=255) 、緑 33(=51) 、青 0 にせよ」という指定を意味する。

宣言ブロックとセレクタを合わせて規則集合という
color: #ff3300 部分を宣言という
宣言の内、: より前（上例では color ）をプロパティ（特性）という
宣言の内、: より後（上例では #ff3300 ）を値という
