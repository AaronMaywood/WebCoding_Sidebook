```{tip}
**高度な話題** 

配付素材の中では`font-size`に単位`rem`を使用しています。しかし、`html { font-size: 62.5%; }`は使用しておりません。

このようなコーディングが便利な時もあります。

これはこの教材のデザインにデザイナーが使用するような`Adobe Photoshop`や`Figma`などのデザインツールを使用しておらず、`CSS`上だけでデザインしたことに由来します。

その際、文字の大きさを「基準の文字の何倍にしようか？」という観点で設計しました。通常の文字の倍にしたければ「`2rem`」と書き、1.8倍にしたければ「`1.8rem`」と書くだけです。

それを、わざわざ「基準の文字を`16px`として、その倍が`32px`、その`1.8`倍が`28.8px`」などと`px`に変換する方がめんどうです。

最初から単位`rem`でデザインしていますので、`px`へと換算する必要がありません。
換算の必要がないので、換算を楽にする `html { font-size: 62.5%; }` のテクニックも必要ないというわけです。
```
