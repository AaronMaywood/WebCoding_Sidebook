````{card} px→rem変換の方法
% 関連: cards/css/62.5.md

`px`を`rem`に変換するには、欲しい`px`数に`1/16`を掛けるだけです。

例)
- 40px は何rem？
  **40** ×1/16 = 2.5rem
- 18px は何rem？
  **18** ×1/16 = 1.125rem

また、`CSS`の電卓機能である`calc()`を使用して、計算をコンピューターにさせることもできます。（`calc()`はレベル8で取り上げます。）

```css
font-size: calc( 40 * 1rem/16 ); /* 40px */
font-size: calc( 18 * 1rem/16 ); /* 18px */
```
````
